Windows Server 2019偏门性问题

发表于2021-11-18|更新于2021-11-23|Windows
|阅读量:

本章主要解决window server 2019的各种偏门问题

DNS

关闭网络掩码排序功能

要求动态更新设置为非安全

证书

颁发的证书有效期修改

环境介绍：CA证书有效期20年，CA颁发证书有效期10年

  1. 修改注册表

  1. 修改默认的web服务器证书

  1. 重启证书颁发机构然后把刚刚创建的web服务器副本添加过来

  1. 提交证书申请测试

  1. 查看证书年限

组策略

拒绝sys组从网络访问域控制器

允许manager组本地登录域控制器

登录时不显示用户名，不显示上次登录，无须按Ctrl+Alt+Del

审核登录事件，同时审核成功和失败

禁用“关闭事件跟踪程序”
