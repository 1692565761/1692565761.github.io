Centos 单用户模式重置密码

发表于2021-11-11|更新于2021-11-12|Linux
|阅读量:

使用单用户模式有一个前提，就是系统引导器（GRUB）能正常工作，进入系统维护就要使用修复模 式，注意：进入单用户模式，没有开启网络服务，不支持远程连接。CentOs 系统中不同的运行级别（Run Leve）代表了系统中不同的运行状态。例如，Linux 服务器正常运行时处于运行级别 3，是能够提供网络 服务的多用户模式：而运行级别 1 只允许管理员通过服务器主机的单一控制台进行操作，即单用户模式。 下面以 GRUB 为例说明其进入方法。在 GRUB 启动菜单中有“E”和“C”操作按键，按 E 键，进入所示 的界面。

使用光标键找到“linux16”下面一行的 ro 符，把 ro 修改为“rw init=/sysroot/bin/sh”


然后按“Ctrl+X”组合键进入单用户模式

chroot，即 change root directory （更改 root 目录）
